## How It Works

The extension injects a content script into YouTube pages. Instead of using `setInterval` or polling, it sets up a `MutationObserver` on `document.body` to watch for DOM changes. When YouTube injects its "Skip Ad" button (using classes like `.ytp-ad-skip-button`, `.ytp-ad-skip-button-modern`, or `.ytp-skip-ad-button`), the observer detects the change and immediately simulates a click.

For unskippable ads, it falls back to seeking the video to the end and muting the audio.
